Overview
========

The Cisco pxGrid DXL client library provides a high level wrapper for
communicating with 
`Cisco pxGrid <https://www.cisco.com/c/en/us/products/security/pxgrid.html>`_
via the
`Data Exchange Layer <https://docs.trellix.com/bundle/data-exchange-layer-4.1.0-product-guide-epolicy-orchestrator/page/GUID-28258D6B-9D48-43CB-994F-1564AD17C545.html>`_
(DXL) fabric.
